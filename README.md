This is my small project.
This project contains the SDL3.dll library and an executable .exe file.
Running the .exe opens a window with three points and three edges.
If you move the mouse and use WASD, the triangle's perspective will change as intended.
The archive contains the same(SDL3.dll & .exe) as the repository.
For the .exe to work, the SDL3.dll library must be in one folder.

Это мой небольшой проект.
В этом проекте есть библиотека SDL3.dll и исполняемый файл .exe.
Если запустить .exe то откроется окно в котором будет 3 точки и три ребра.
Если двигать мышкой и перемещатся на WASD то перспектива треугольника будет менятся как и было это задуманно.
В архиве тоже самое что и в репозитории
Для работы .exe в одной папке должна быть библеотека SDL3.dll
